# Primitive types

- typeof // Kiểm tra loại kiểu dữ liệu
- Type Assignment
  `const variable: dataTypes = value`
- Number Types
  `const age:number = 12`
- String Types
  `const name:string = 'son'`
  or
  `const name = 'son'`
- Boolean Types
  `const name:boolean = false`

_Note:_ - optional: không bắt buộc

```php
    name?: parameter này không bắt buộc
```
